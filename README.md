## Structure

```
├── Arrays/
├── Strings/
├── LinkedList/
├── Trees/
├── Graphs/
├── DynamicProgramming/
├── Greedy/
├── Backtracking/
├── BitManipulation/
├── Miscellaneous/
└── progress.md          # daily log — one line per day
```

Each topic folder contains a `solutions/` directory. Every solution file follows the naming pattern:

```
DD-MM-YYYY_ProblemName.java
```

And includes a short header comment block (problem link, approach, time/space complexity) — see `TEMPLATE.java` in this repo root for the format.

## How I log a day

1. Solve the problem in the relevant topic folder under `solutions/`
2. Add one line to `progress.md`
3. Commit with a message like: `Day 12: Solved 'Longest Substring Without Repeating Characters' (Sliding Window)`
